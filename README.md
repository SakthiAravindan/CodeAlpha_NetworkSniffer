CodeAlpha Task 1: Basic Network Sniffer

This is a simple Python-based network sniffer built for the CodeAlpha Cybersecurity Internship.

 Features
- Captures live network packets
- Displays:
  - Source IP
  - Destination IP
  - Protocol used (TCP, UDP, etc.)

- Scapy Library
